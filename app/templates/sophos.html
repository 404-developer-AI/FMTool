Setup Guide: Enable Sophos XGS API

Follow these steps to enable the REST API on your Sophos XGS firewall.

1
Log in to the Sophos XGS web console

Open https://<firewall-ip>:4444 in your browser and log in with an administrator account.

2
Navigate to API settings

Go to Backup & Firmware > API (or Administration > API depending on your firmware version).

3
Enable API access

Toggle the API Configuration switch to enable the REST API.

4
Set allowed IP address

Add the IP address of the machine running FMTool to the allowed source IP list. This restricts API access to your workstation.

5
Create an API admin user

Go to Authentication > Users and create a dedicated user with Administrator privileges for API access.

6
Configure FMTool credentials

Copy config.local.example.py to config.local.py in the FMTool root directory and fill in your credentials:

SOPHOS_HOST = "192.168.1.1"
SOPHOS_USERNAME = "api-admin"
SOPHOS_PASSWORD = "your-password"
SOPHOS_PORT = 4444
7
Restart FMTool

Stop and restart the FMTool application to load the new credentials. Then use the Test Connection button above to verify.

Rate limiting: The Sophos API is rate-limited. FMTool handles this automatically with retry logic, but avoid running multiple instances simultaneously.
